## **Prompt 20 — RFQ for Warehouse Management System**

Backstory:

 A distribution company needs a robust Warehouse Management System (WMS) to handle inventory tracking, order processing, and reporting. They are looking for vendors with experience in integrating WMS into existing ERP systems.

Goal:

 Create an RFQ for WMS software vendors.

Prompt:

 *"You are the operations IT head drafting an RFQ for WMS providers. Your email should:*

  - Outline warehouse operations and system requirements

  - Request features like barcode scanning, order tracking, and reporting

  - Ask for integration capability with current ERP

  - Include pricing models, licensing details, and training support

  - Provide RFQ submission deadline"

Inputs Required:

  - Current warehouse operations description

  - Required WMS features

  - ERP integration needs

  - Number of warehouse locations/users

  - Budget range

  - RFQ submission date
