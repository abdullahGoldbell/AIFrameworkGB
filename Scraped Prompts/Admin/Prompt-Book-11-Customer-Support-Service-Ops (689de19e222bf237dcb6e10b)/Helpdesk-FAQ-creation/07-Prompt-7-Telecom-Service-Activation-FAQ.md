## **Prompt 7 — Telecom Service Activation FAQ**

Backstory:

 A 31-year-old customer care manager at a Singapore telecom provider headquartered in the CBD is struggling with repeated calls about SIM activation, porting, and eSIM setup. She wants an FAQ that answers these queries in plain, jargon-free language.

Goal:

 Build a customer-friendly telecom service activation FAQ.

Prompt:

 "You are a telecom helpdesk writer. Prepare an FAQ covering:

  - New SIM activation process

  - Number portability steps (per IMDA guidelines)

  - eSIM activation

  - Common errors and their solutions"

Inputs Required:

  - Telecom activation process docs

  - IMDA regulatory timelines for porting

  - Error code list with fixes
