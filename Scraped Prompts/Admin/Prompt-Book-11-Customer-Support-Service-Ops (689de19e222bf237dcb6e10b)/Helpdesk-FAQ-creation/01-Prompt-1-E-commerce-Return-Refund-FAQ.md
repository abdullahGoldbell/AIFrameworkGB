## **Prompt 1 — E-commerce Return & Refund FAQ**

Backstory:

 A 27-year-old operations manager at a Singapore-based online fashion store headquartered in Paya Lebar is struggling with repetitive queries about returns and refunds. Customers often get frustrated because they can't find clear answers on the website, leading to unnecessary calls to customer care. She wants a comprehensive FAQ page that explains timelines, conditions, and refund modes in plain English (with optional Mandarin and Malay versions for broader reach).

Goal:

 Create a customer-friendly FAQ for returns and refunds.

Prompt:

 "You are an e-commerce helpdesk content specialist. Create a multilingual (English + Mandarin + Malay) FAQ section for returns and refunds.

 Include:

  - Return eligibility criteria

  - Step-by-step return request process

  - Refund timeline and mode of refund

  - Common exceptions (e.g., clearance items)

Inputs Required:

  - Return policy document

  - Refund timelines

  - List of exceptions or special cases

  - Preferred tone (formal, casual, etc.)
