## **Prompt 9 —  Delivery Confirmation – English & Punjabi**

Backstory:

 A Singapore-based organic food brand delivers across the island and serves a notable Punjabi-speaking customer segment in the Sikh community around Silat Road and beyond. Customers often call to ask if delivery has been completed. A standardized bilingual confirmation message would improve efficiency and trust.

Goal:

 Create a bilingual delivery confirmation script that thanks customers and confirms order completion.

Prompt:

 "You are a last-mile delivery communication expert. Create a bilingual (English & Punjabi) delivery confirmation script that includes:

  - Thank you message with customer’s name

  - Order details (short)

  - Confirmation of delivery date/time

  - Feedback request link"

Inputs Required:

  - Customer name

  - Order ID

  - Delivery date/time

  - Feedback link
