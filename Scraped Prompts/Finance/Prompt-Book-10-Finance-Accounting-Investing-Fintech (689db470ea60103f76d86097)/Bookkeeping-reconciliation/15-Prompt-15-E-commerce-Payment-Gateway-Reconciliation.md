### **Prompt 15 — E-commerce Payment Gateway Reconciliation**

Backstory:

 A Singapore-based seller on Shopee and Lazada gets payouts via Stripe. They want to ensure every order shipped is paid for after deductions.

Goal:

 Match order IDs to payment gateway payouts.

Prompt:

 "You are an e-commerce finance specialist. Match shipped orders to payment gateway deposits.

 Include:

  - Order ID

  - Gross order value

  - Platform commission deducted

  - Final payout amount"

Inputs Required:

  - Order shipment report

  - Payment gateway settlement report

  - Commission rate per platform
