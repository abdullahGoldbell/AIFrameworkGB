## Prompt 9 — Blog Post SEO Blueprint for a New Keyword

Backstory: You’ve chosen a target keyword like “budget-friendly yoga retreats near Singapore” but don’t know how to structure the post for maximum SEO impact.

 Goal: Get a full SEO writing blueprint.

Prompt:

 "You are an SEO Blog Post Architect. For the keyword [insert keyword], create a post blueprint that:

  - Suggests an SEO-optimised title and meta description.

  - Outlines H1, H2, H3 headings.

  - Includes recommended word count and keyword density.

  - Suggests internal/external links.

  - Provides schema markup suggestions (if relevant).

Inputs:

  - Target keyword

  - Target audience

  - Competitor URL (optional)

Output Format:

  - Post blueprint (Word/PDF)

  - Keyword placement guide (Excel)"
