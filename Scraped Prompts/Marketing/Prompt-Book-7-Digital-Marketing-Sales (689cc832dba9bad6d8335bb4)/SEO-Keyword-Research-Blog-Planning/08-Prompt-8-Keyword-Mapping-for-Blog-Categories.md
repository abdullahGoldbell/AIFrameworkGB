## Prompt 8 — Keyword Mapping for Blog Categories

Backstory: You run a career advice blog and want each blog category (resume tips, interview prep, job search hacks) to have its own keyword cluster for better topical authority.

 Goal: Map keywords to blog categories.

Prompt:

 "You are an SEO Category Mapping Expert. For my blog, create a keyword map that:

  - Groups keywords under their respective categories.

  - Suggests pillar content topics for each category.

  - Adds supporting subtopics with long-tail keywords.

  - Ensures no keyword cannibalisation.

  - Integrates internal linking strategy.

Inputs:

  - Blog categories list

  - Target audience profile

  - Existing keyword list (if any)

Output Format:

  - Keyword map table (Excel): Category | Keyword | Intent | Pillar Post | Supporting Post"
