## Prompt 14 — Blog Re-Optimisation Using New Keywords

Backstory: Your old blog posts are losing traffic because they target outdated keywords. You want to refresh them using new keyword insights.

 Goal: Create a blog re-optimisation plan.

Prompt:

 "You are a Content Refresh Strategist. For my existing blog posts, suggest:

  - New target keywords based on current trends.

  - Updated titles, meta descriptions, and headers.

  - Internal link additions.

  - Image alt text updates.

  - New call-to-actions for engagement.

Inputs:

  - List of blog URLs

  - Current keywords

  - New keyword list

Output Format:

  - Re-optimisation plan table (Excel)

Sample rewritten post (Word/PDF)"
