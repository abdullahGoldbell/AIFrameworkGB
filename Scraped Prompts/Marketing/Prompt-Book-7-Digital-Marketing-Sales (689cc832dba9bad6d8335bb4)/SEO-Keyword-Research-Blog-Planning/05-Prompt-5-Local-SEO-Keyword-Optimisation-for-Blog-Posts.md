## Prompt 5 — Local SEO Keyword Optimisation for Blog Posts

Backstory: You run a blog about weekend getaways from Singapore (short-haul trips to Johor Bahru, Bintan, Batam, Penang, KL, Phuket, Bali, plus Singapore staycations) and want to target city/destination-specific audiences for better local search ranking.

 Goal: Create a local SEO keyword list optimised for multiple Southeast Asia destinations popular with Singapore travellers.

Prompt:

 "You are a Local SEO Keyword Strategist. For my travel blog, create a local SEO plan that:

  - Identifies 5–10 high-volume destination-specific keywords per location.

  - Suggests modifiers for local search intent (e.g., “best resorts in [Destination] from Singapore”).

  - Shows how to integrate these into meta titles, descriptions, and headers.

  - Prioritises destinations with high travel search demand from Singapore.

  - Suggests content formats that perform well in local SEO.

Inputs:

  - List of target destinations (e.g., Sentosa, JB, Bintan, Batam, Penang, KL, Bali)

  - Travel niches covered (e.g., resorts, hiking trails, heritage spots, family-friendly)

  - Competitor blogs targeting these destinations

Output Format:

  - Local keyword map (Excel)

  - On-page SEO placement guide (Word/PDF)"
