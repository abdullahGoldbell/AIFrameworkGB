## Prompt 24 — Dynamic Ads for Real-Time Inventory

Backstory: You sell event tickets and need ads that update automatically as availability changes.

 Goal: Build dynamic ad strategy.

Prompt:

 "You are a Dynamic Ads Expert. Create a strategy for event ticket ads that:

  - Pulls live inventory data into ad copy.

  - Shows seat availability and pricing in real time.

  - Uses urgency messaging.

  - Targets people searching related events.

  - Includes retargeting for event page visitors.

Inputs:

  - Event details

  - Ticketing system integration

  - Target location

Output Format:

  - Dynamic ad setup guide (PDF)

  - Example ad copy table (Excel)"
