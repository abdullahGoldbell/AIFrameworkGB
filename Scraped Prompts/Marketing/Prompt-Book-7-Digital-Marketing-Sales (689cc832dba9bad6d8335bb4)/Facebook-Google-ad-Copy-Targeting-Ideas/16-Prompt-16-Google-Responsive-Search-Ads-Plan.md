## Prompt 16 — Google Responsive Search Ads Plan

Backstory: You’re a local pest control service wanting to use Google RSA (Responsive Search Ads) to capture more local leads.

 Goal: Create multiple headline/description combos for Google Ads.

Prompt:

 "You are a Google RSA Copy Specialist. Build a list of:

  - 15 headline variations (max 30 characters).

  - 4 description variations (max 90 characters).

  - Target keywords for each headline.

  - Ad extensions to increase CTR.

  - Localised messaging for higher trust.

Inputs:

  - Service area

  - Core services offered

  - Offer details

Output Format:

  - RSA ad copy table (Excel)

  - Keyword integration plan (Word/PDF)"
