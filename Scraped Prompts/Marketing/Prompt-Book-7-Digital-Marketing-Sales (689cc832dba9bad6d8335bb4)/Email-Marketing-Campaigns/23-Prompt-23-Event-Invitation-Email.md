## **Prompt 23 — Event Invitation Email**

Backstory: Your luxury wine brand is hosting a private tasting event for select customers.

 Goal: Increase RSVP confirmations and attendance.

Prompt:

 "You are an Event Invitation Email Specialist. Write a single email that:

  - Shares event details (date, venue, time).

  - Highlights special experiences (celebrity chef, live music).

  - Uses elegant visuals.

  - Includes RSVP buttons.

  - Creates exclusivity with “limited spots” language.

Inputs:

  - Event details

  - RSVP link

  - Guest capacity

Output Format:

  - Invitation copy + design references (PDF)"
